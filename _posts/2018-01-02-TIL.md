# HTML syntax

```
<html>
    <head>
    </head>
    <body>
        <p> This is a paragraph </p>
    </body>
</html>
```

# 개발 환경
- Browser : Chrome
- Editor : MS Code

# Trees
- element1은 element2/3의 Parent
- element2와 element3은 siblings

HTML에서, 
```
<div>
    <h1> Article Title </h1>
</div>
```
h1 is a child of div

```
<div>
    <h1> Article Title </h1>
    <p> Paragraph </p>
</div>
```
h1 and p is a sibling
and div is a parent of h1 and p
