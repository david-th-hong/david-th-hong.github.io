## CSS

### CSS Rulesets

```
 div {
     text-align: right;
 }
```
 
 div : selector

 {...} : declaration block

```
<style> 
 p {
     color: blue;
 }
 /* Comment in CSS*/
</style>

<!-- Comment in HTML -->
```

### Attribute and Selector

* Tag Selector
```
h1 {
    color: green;
}
```

* class Attribute Seletor
```
.book-summary {
    color: red;
}
```

* id Attribute Selector
```
#site-description{
    color: red;
}
```
