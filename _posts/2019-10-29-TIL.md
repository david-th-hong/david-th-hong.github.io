### 개발환경구성

> **tensorflow**
>
> - python3.6 설치 (64bit)
> - pip3 설치
> - virturalenv 설치
> - venv 환경 구성
> - tensorflow 설치

```bash
❯ virtualenv -p python3.6 tensorflow //tensorflow venv 설정(python 3.6버전)
❯ source ./tensorflow/bin/activate //tensorflow venv 진입
tensorflow ❯ pip install --upgrade tensorflow // pip로 tensorflow 설치
```
